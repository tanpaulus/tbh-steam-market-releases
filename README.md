# TBH Steam Market — Downloads

**[⬇ Download the latest version](../../releases/latest)** — grab the file named `TBH Steam Market Setup x.x.x.exe`.

## Install
1. Download and run the **Setup** `.exe`.
2. Windows may show **"Windows protected your PC"** — this is the normal warning for free apps that aren't code-signed, not a problem with this app. Click **"More info" → "Run anyway"**.
3. Done! The app updates itself automatically whenever a new version is released.

> Use the **Setup** installer (not the `portable` `.exe`) so auto-update works.
